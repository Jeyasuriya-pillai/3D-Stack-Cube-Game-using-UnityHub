# 🎮 Stack Cube Game (Unity)

A simple 3D stack cube game built using Unity and C#. The player must stack moving cubes perfectly to build the tallest tower.

## 🚀 Features
- Smooth stacking mechanics
- Increasing difficulty
- Minimal 3D design

## 🛠️ Tech Stack
- Unity Engine
- C#

## 🎮 Gameplay
Click to drop cubes and stack them accurately.
